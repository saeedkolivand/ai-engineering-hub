Why it's different

Built for operators, not dashboards.

GitHub, Linear, and Datadog density — with Apple-grade restraint. An operational tool, not a KPI marketing screen.

Dynamic sources

Tools register themselves. No enum to edit, no rebuild to ship a new collector.

Dimensional analytics

Group every metric by source, provider, agent, and repository — and trust the dashes.

Drill down anywhere

Repository → Session → Task → Agent, tables-first. Press ⌘K to jump straight to it.

Stream Deck plugin

Nine live hardware monitors driven by the same API. No log parsing on the device.

One process

A Rust core — Axum, SQLx, SQLite — runs inside Tauri. No sidecars, nothing to deploy.

Themed from tokens

Light, dark, and system — the desktop UI and the plugin stay consistent automatically.

Quick start

Running in under a minute.

Install, then start the desktop app — UI, Hub API, and collectors in one process. Enable your tools and data flows in within about five seconds.
